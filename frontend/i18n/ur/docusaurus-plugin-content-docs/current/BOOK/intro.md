# تعارف

**فزیکل اے آئی اور ہیومنائیڈ روبوٹکس** پر جامع گائیڈ میں خوش آمدید۔

یہ کتاب آپ کو ایک ابتدائی سے ذہین جسمانی نظاموں کی تعمیر میں ماہر بنانے کے لیے ڈیزائن کی گئی ہے۔

### آپ کیا سیکھیں گے:
* **اعصابی نظام:** ROS 2 اور روبوٹ مواصلات۔
* **ڈیجیٹل ٹوئن:** NVIDIA Isaac Sim میں اعلیٰ معیار کی نقل۔
* **دماغ:** فیصلہ سازی کے لیے Vision-Language-Action (VLA) ماڈلز۔



---
:::tip ہارڈویئر تیار ہے؟
یقینی بنائیں کہ آپ کے پاس نقل کے ابواب کی پیروی کرنے کے لیے **NVIDIA RTX 4070 Ti** یا بہتر ہے۔
:::
