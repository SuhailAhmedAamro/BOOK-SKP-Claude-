# ٹیوٹوریل کا تعارف

آئیے **5 منٹ سے کم میں Docusaurus** کو دریافت کریں۔

## شروعات کریں

**نئی سائٹ بنا کر** شروعات کریں۔

یا **[docusaurus.new](https://docusaurus.new)** کے ساتھ **Docusaurus کو فوری طور پر آزمائیں**۔

### آپ کو کیا ضرورت ہوگی

- [Node.js](https://nodejs.org/en/download/) ورژن 20.0 یا اس سے اوپر:
  - Node.js انسٹال کرتے وقت، تمام انحصار سے متعلق چیک بکس کو چیک کرنے کی سفارش کی جاتی ہے۔

## نئی سائٹ بنائیں

**classic template** استعمال کرتے ہوئے نئی Docusaurus سائٹ بنائیں۔

کمانڈ چلانے کے بعد classic template خودکار طور پر آپ کے پروجیکٹ میں شامل ہو جائے گا:

```bash
npm init docusaurus@latest my-website classic
```

آپ یہ کمانڈ Command Prompt، Powershell، Terminal، یا اپنے کوڈ ایڈیٹر کے کسی دوسرے مربوط ٹرمینل میں ٹائپ کر سکتے ہیں۔

یہ کمانڈ Docusaurus چلانے کے لیے ضروری تمام انحصار بھی انسٹال کرتا ہے۔

## اپنی سائٹ شروع کریں

ڈویلپمنٹ سرور چلائیں:

```bash
cd my-website
npm run start
```

`cd` کمانڈ اس ڈائریکٹری کو تبدیل کرتا ہے جس کے ساتھ آپ کام کر رہے ہیں۔ اپنی نئی بنائی گئی Docusaurus سائٹ کے ساتھ کام کرنے کے لیے، آپ کو وہاں ٹرمینل کو نیویگیٹ کرنا ہوگا۔

`npm run start` کمانڈ آپ کی ویب سائٹ کو مقامی طور پر بناتا ہے اور اسے ڈویلپمنٹ سرور کے ذریعے پیش کرتا ہے، جو http://localhost:3000/ پر دیکھنے کے لیے تیار ہے۔

`docs/intro.md` (یہ صفحہ) کھولیں اور کچھ لائنیں ترتیب دیں: سائٹ **خودکار طور پر دوبارہ لوڈ** ہوتی ہے اور آپ کی تبدیلیاں دکھاتی ہے۔
